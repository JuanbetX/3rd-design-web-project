# 3rd-design-web-project
# Silent Hill 2 — Sitio académico

Breve sitio temático para el proyecto final del curso de Diseño Web. Incluye tres páginas enlazadas, estilos en CSS, elementos multimedia y publicación mediante GitHub Pages.

## Tecnologías
- HTML5 semántico (header, nav, main, section, footer)
- CSS3 (tipografía, paleta, layout responsive)

## Estructura
- `index.html` (inicio, hero, trailer)
- `historia.html` (premisa, personajes, simbolismo)
- `galeria.html` (imágenes y enlaces externos)
- `css/style.css`
- `img/` (logo, hero, galería)
- `video/` (trailer opcional)
